# Instagram Reel Downloader

A simple Flask web application to download Instagram Reels.

## Online Version

If you don't want to host it yourself, use the online version at https://instagram.renato.ink/ - 100% free and without ads.

## Local Installation

### Prerequisites

- Python 3.7 or higher

### Setup

1. Clone the repository:
```bash
git clone https://github.com/renvto/instagram-reels-downloader.git
```

2. Navigate to the project directory:
```bash
cd instagram-reels-downloader
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Run the application:
```bash
python main.py
```

5. Open your browser and go to the local address shown in the terminal (usually http://127.0.0.1:5000)

## How to Use

1. Copy an Instagram Reel link
2. Paste it into the input field
3. Click the Download button
4. The video will be downloaded to your device
